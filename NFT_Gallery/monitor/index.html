üöÄ Quick Start Commands

Use these commands to interact with the NFT Downloader:

# Download NFTs from a wallet docker-compose run --rm nft-downloader --wallet YOUR_WALLET_ADDRESS # Validate configuration docker-compose run --rm nft-downloader --wallet YOUR_WALLET_ADDRESS --validate-only # Show statistics docker-compose run --rm nft-downloader --wallet YOUR_WALLET_ADDRESS --stats # With custom output directory docker-compose run --rm nft-downloader --wallet YOUR_WALLET_ADDRESS --output /app/output/custom

üìÅ Directory Structure

The following directories are mounted as volumes:

  • ./output/ - Downloaded NFT images
  • ./logs/ - Application logs
  • ./credentials/ - Google Cloud credentials

üîß Configuration

Make sure you have configured the following:

  1. Set GOOGLE_CLOUD_PROJECT in your .env file
  2. Place your Google Cloud service account key in credentials/service-account.json
  3. Ensure your Ankr API key is stored in Google Secret Manager
